# ✔ minlog
>minhyung + blog의 합성어로 CRUD 게시판 형식의 개인 블로그 API 프로젝트입니다.

<br>

## 개발기간 및 참여인원
- 2022.07.02 ~ 2022.07.19
- 개인 프로젝트

<br>

## 사용 기술
- Java 11 (jar)
- Springboot 2.7.1
- Gradle
- Spring Data JPA
- QueryDSL
- H2 Database
- Vue.js 3.0
- Element UI

<br>
